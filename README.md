# 星穹铁道工具箱
星穹铁道工具箱是一个用于修改画质和帧率的工具箱。它可以让你更好地享受星穹铁道游戏。

# 功能
### 当前版本的工具箱包括以下功能：
帧率解锁

画质修改(未实现)

等等未实现的功能...

# 使用方法
下载本工具箱的最新版本。

解压缩下载的文件。

运行工具箱，并按照提示进行操作。

# 注意事项
本工具箱仅适用于PC版星穹铁道游戏。

修改游戏画质和帧率可能会影响游戏的稳定性和性能。

在使用本工具箱前，请备份游戏文件以防数据丢失。

使用本工具箱可能违反游戏开发者的使用协议，使用风险自负。
